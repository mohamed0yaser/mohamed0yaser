A feature-rich Movies Application built with Flutter and powered by TMDb API. The app allows users to discover, search, and explore movies and TV shows with detailed information, ratings, and recommendations. Designed with a clean UI, real-time data, and full API integration.

Core Features

Movie & TV Search

Search any movie, TV show, or actor with instant results powered by TMDb.

Ratings & Reviews

Displays IMDb & TMDb ratings with user reviews and popularity scores.

Categories & Genres

Browse by trending, upcoming, top-rated, or genre-based movie lists.

Trailers & Media

Watch official trailers, teasers, and clips directly from YouTube/TMDb links.

Cast & Crew

Detailed profiles of actors, directors, and production teams.

Similar & Recommended

AI-powered recommendations for similar movies & shows.

Tech Stack

Flutter (Dart)

Cross-platform development for iOS, Android, and Web.

TMDb API

The Movie Database API powering all content & metadata.

YouTube API

For streaming trailers and official media content.

Bloc / Cubit

Efficient state management for scalable architecture.

Project Goals

This project demonstrates my ability to integrate complex APIs into real-world applications. It focuses on creating a cinema-like browsing experience, combining real-time data, beautiful UI, and smooth navigation to help users explore entertainment content easily.

This Movies App reflects my skills in API integration, state management, and UI/UX design, delivering a seamless and interactive movie discovery experience.
